Qui somme nous?

Le festival Les Beaux Bagages a été créé pour rapprocher la culture des citoyens du Creusot, tout au long de l'été, nous aurons différents types d'attractions répartis dans toute la ville! N'oubliez pas de le vérifier!

Quoi nous propose?

Musique

Plusieurs artistes vous apportent le meilleur de la musique!

Théâtre

Plusieurs œuvres théâtrales pour apporter toutes les émotions les plus incroyables aux spectateurs!

Cirque

Les meilleurs spectacles que vous puissiez imaginer!

Et beaucoup plus!
